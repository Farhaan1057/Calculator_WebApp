# 🧮 Modern Web Calculator

A feature-rich calculator web application built with HTML/CSS/JS. Designed for both desktop and mobile use with a focus on clean UI and smooth user experience.

## ✨ Features

- Basic arithmetic operations (+, -, ×, ÷)
- Decimal and percentage calculations
- Clear and backspace functions
- Keyboard input support (numbers and operators)
- Dark/Light theme toggle
- Responsive design for all screen sizes
- Error handling (division by zero, invalid input)

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (Vanilla)
